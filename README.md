# Git / Terminal 

본 저장소는 Git을 사용하여 버전 관리를 진행했으며, GUI(SourceTree)와 함께 터미널 기반 Git 명령어를 병행하여 사용하고 있습니다.

## 명령어
```bash
# 현재 경로 및 파일 확인
pwd
ls -a

# Git 상태 확인
git status

# 변경 사항 스테이징 및 커밋
git add .
git comit -m "Singly Linked List"

# 원격 저장소로 푸시
git push
```
